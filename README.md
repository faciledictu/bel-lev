## How to use

| Command                   | Action                                           |
| :------------------------ | :----------------------------------------------- |
| `npm install`             | Installs dependencies                            |
| `npm run dev`             | Starts local dev server at `localhost:4321`      |
| `npm run build`           | Build the site to `./dist/`          |
| `npm run preview`         | Preview the build locally, before deploying     |
